The Secret Life of Letters
Codename: Cipher · Companion sheet: Codebreaker’s Kit
ObjectiveHide a message so no one can read it — then break one that someone tried to hide from you.

Briefing

Long before radios, every secret came down to scrambling letters. Today you build the tool the spies used, learn the weapon the codebreakers used against it, and crack a real intercept. Then you meet the one code that can never be broken. Your printed Codebreaker’s Kit is the hands-on half of this mission — keep it beside you.

Field Operations

  1. Build the cipher wheel from the Kit. Send each other messages with a secret shift.
  2. Learn frequency analysis: count the letters, the most common one is probably a hidden E.
  3. Crack the first intercept in the Kit using only the frequency trick. Report the decoded order to your Handler.
  4. Write the Python Caesar-cracker together — watch a computer break in milliseconds what you just did by hand.
  5. Try a one-time pad: a random key, as long as the message, a different shift for every letter.
Shannon’s twist. Your Caesar cipher breaks because each letter always hides the same way. Shannon proved the one-time pad has perfect secrecy — truly unbreakable — but only if you and your partner share the exact same secret key. That key-sharing problem is the hardest part of every secret in history.

Equipment

  • Printed Codebreaker’s Kit
  • Scissors + a brad (split pin)
  • Laptop with Python

Debrief — mission complete when

  • You cracked the intercept by hand
  • The Python cracker runs
  • You can explain the one-time pad’s catch
Handler note: Resist giving the shift away. The struggle is the lesson — the reveal is on the Kit for emergencies only.
