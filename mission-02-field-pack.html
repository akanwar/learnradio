See the Invisible
Field Pack · RTL-SDR Quick-Start
ObjectiveTurn a $30 USB stick and your laptop into a window onto every radio wave in the air around your house.

Gear check

An RTL-SDR dongle (the RTL-SDR Blog V4 kit is the standard pick — it comes with the dipole antenna you’ll need), its little antenna on the magnetic base, and a laptop. The dongle only listens, so no license is needed for this mission.

Setup — do it once, with your Handler

  1. Download SDR++ (the easiest modern receiver). For the V4 dongle, grab the nightly build — older versions don’t know about the V4 yet.
  2. Plug the dongle into a USB port. Don’t use a flimsy extension — a weak port causes wavy lines and no signals.
  3. Windows only: run Zadig. Choose Options → List All Devices, select Bulk-In, Interface (Interface 0), and click Install / Replace Driver to load WinUSB. (Mac/Linux skip this step.)
  4. Open SDR++. Set the Source to RTL-SDR, then press the big Play button.
  5. Slowly raise the Gain slider until signals appear but the noise doesn’t turn to mush. Tune by clicking on the waterfall.
The #1 V4 gotcha: if you see nothing, or stations show up at the wrong frequency, you’re using the old drivers. The V4 needs the RTL-SDR Blog drivers to work at all — redo the Zadig step. On Windows 11 you may also need to turn off Core Isolation → Memory Integrity, which blocks the driver.

The noise floora quiet gapSteady hiss = pure noise
Shannon’s idea, made visible. On the waterfall, a real signal is a bright stripe standing above the fuzzy background. That background is noise, and the height of the stripe above it is the signal-to-noise ratio. Shannon proved this ratio sets the speed limit for how many bits a channel can carry — the cleaner the signal stands out, the more it can say. You’ll put a real number on this in Mission 04.
Safety: the SDR only receives — it never transmits, so it’s perfectly legal before you’re licensed. Treat the tiny antenna gently and keep the dongle away from anything hot; they warm up in use.

How to send it

A dash lasts about as long as three dots. Leave one dot’s worth of silence between beeps, three between letters, and seven between words. Tap it on your Mission 02 buzzer, or just hum it.

What the graph shows. The line should sag as you walk away — signal gets weaker with distance. Find where it sags below where messages stop arriving: that’s your link’s edge. Now raise the spreading factor and the edge moves farther out, but each message takes longer to send. That trade is Shannon’s channel-capacity law, drawn in your own pencil.
